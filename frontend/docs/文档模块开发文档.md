# 文档模块开发文档

> 开发时间: 2026-05-25
> 版本: 1.0.0

## 概述

文档模块是一个独立于博客系统的完整文档管理系统，支持树形结构的文档组织、多文档书管理、全文搜索等功能。

## 功能特性

- **文档书管理** - 创建多个独立的文档集合
- **树形文档结构** - 支持多级嵌套的文档层级
- **文档状态管理** - 草稿/已发布/隐藏三种状态
- **全文搜索** - 基于标题和内容的文档搜索
- **文档导航** - 上一篇/下一篇/面包屑导航
- **权限控制** - 支持公开/私密文档书
- **主题支持** - 可配置的文档主题样式

## 使用示例

### 创建文档书

```typescript
import { docBookApi } from '@/api'

const docbook = await docBookApi.create({
  name: 'API 文档',
  slug: 'api-docs',
  description: '项目 API 接口文档',
  is_public: true,
  theme: 'default'
})
```

### 创建文档

```typescript
import { docApi } from '@/api'

const doc = await docApi.create({
  docbook_id: 1,
  title: '快速开始',
  slug: 'getting-started',
  content: '# 欢迎使用\n\n这是第一篇文档...',
  parent_id: 0,
  status: 'published'
})
```

### 获取文档树

```typescript
import { docApi } from '@/api'

const tree = await docApi.getTree(docbookId)
// 返回树形结构，可直接用于渲染侧边栏导航
```

### 搜索文档

```typescript
import { docApi } from '@/api'

const results = await docApi.search(docbookId, '用户认证', 20)
// 搜索标题和内容中包含"用户认证"的文档
```

## 注意事项

1. **slug 唯一性**
   - 文档书的 slug 在全局必须唯一
   - 文档的 slug 在同一文档书内必须唯一

2. **树形结构**
   - parent_id = 0 表示顶级文档
   - level 自动根据父文档计算
   - path 自动维护，格式如 "0/1/2"

3. **级联删除**
   - 删除文档书会级联删除所有文档
   - 删除文档会级联删除所有子文档

4. **权限控制**
   - 只有文档书作者可以编辑/删除
   - 公开文档书所有人可访问
   - 私密文档书仅作者可访问
